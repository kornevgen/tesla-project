\paragraph{Постановка задачи}

есть тестовый шаблон для микропроцессора некоторой архитектуры, для
него надо сгенерировать тестовую программу

\paragraph{Результаты}

Основные научные результаты, полученные в рамках диссертационной
работы, состоят в следующем:

\begin{enumerate}
\item предложена модель модулей управления памятью ?
\item предложен метод генерации ограничений (т.н. \emph{совместная
генерация ограничений}) для тестовых шаблонов, нацеленных на
тестирование инструкций обращения к памяти;
\item предложен метод \emph{зеркальной генерации} ограничений для тестовых
шаблонов, нацеленных на тестирование инструкций обращения к памяти;
\item предложен метод описания стратегий вытеснения в кэш-памяти в виде
ограничений \emph{перебором диапазонов вытеснения};
\item предложен метод описания стратегий вытеснения в кэш-памяти в виде
ограничений \emph{с использованием функций полезности}.
\end{enumerate}

\paragraph{Эксперименты}

MIPS

модели удовлетворяют MMU у MIPS, PowerPC, Alpha и Pentium.
