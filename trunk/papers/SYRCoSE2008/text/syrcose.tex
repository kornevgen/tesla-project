\begin{abstract}
%–ассматриваетс€ задача генерации тестовых данных дл€ тестировани€
%арифметической подсистемы центральных процессоров \textsc{MIPS64}.
%ƒл€ решени€ этой задачи предлагаетс€ метод, использующий формальное

The main problem is test data generation for arithmetic subsystem
testing of CPUs \textsc{MIPS64}. A method is proposed to solve this
problem. The method uses a formal description of processor
functionality. A language for this description is proposed also.
This language permits automatic test data generation.
\end{abstract}
